The fermata orientation object

Type: String

The visual orientation of a fermata glyph, with respect to its corresponding notation.

Allowed values:

Value Description
"above" The fermata should be placed above its corresponding notation.
"auto" Consuming applications are free to use their own algorithms to decide the fermata's vertical placement.
"below" The fermata should be placed below its corresponding notation.

Parent objects
