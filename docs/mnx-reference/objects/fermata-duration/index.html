The fermata duration object

Type: String

These values represent the duration of a fermata's pause.

Allowed values:

Value Description
"auto" Consuming software is free to determine the duration according to its own algorithms.
"long"
"none" The fermata doesn't affect playback. (In other words, playback should proceed as if the fermata didn't exist.)
"normal"
"short"
"veryLong"
"veryShort"

Parent objects

This object is used by the following parent objects:

  • fermata: "duration"
